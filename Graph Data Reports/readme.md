### Guide on Graph Reports

In this folder you will find the scores for each model, for each question. You will also see the stats sheets used to average the performance latencies of the models including the token usage. 
Note that for "correct" the entry in a cell is 1 (T) and for "incorrect" the entry in a cell is 0 (F). 

For anonymity, we perform a text replacement of items such as room names with phonetic alphabet (e.g., Alpha, Beta, Charlie, etc).
